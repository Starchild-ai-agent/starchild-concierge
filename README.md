# Starchild Concierge

Knowledge base for the Starchild welcome agent.

## Purpose

This repository contains the knowledge base used by Starchild's concierge agent—the AI assistant that greets new users and answers questions before they create their first agent.

## Files

- **`welcome-knowledge.md`** - Main knowledge base (the concierge reads this)
- **`opening-lines.md`** - Tested greeting variations
- **`llms-updates.md`** - Suggested additions to the main LLMS.txt
- **`changelog.md`** - Version history

## Usage

The concierge agent loads `welcome-knowledge.md` as its primary knowledge source. Updates to this file are reflected immediately.

## Tone

Calm, spacious, futuristic, all-knowing. Not a chatbot—a cosmic intelligence.
