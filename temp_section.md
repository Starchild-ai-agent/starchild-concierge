### What Users Can Do

**Everything OpenClaw can do:**
- Chat, research, analyze documents
- Write and review code
- Manage files, run scripts
- Set up automations and scheduled tasks
- Connect to APIs and external services

**Plus trading superpowers:**

Starchild adds custom-built tools for controlling your finances:

**Trading:**
- Perpetual futures (Orderly, HyperLiquid)
- Cross-chain swaps (WooFi: EVM + Solana)
- Automated strategies ("Vibe Trading" - describe idea, agent builds it)
- DeFi protocols, prediction markets (Polymarket)
- Funding rate monitoring, order book analysis
- Backtesting before going live

**Research:**
- Web, social, on-chain data
- Sentiment tracking across markets
- Wallet monitoring, smart money flows
- Narrative momentum detection

**Automation:**
- Recurring trades, alerts, position management
- Risk monitoring (stop losses, take profits)
- Portfolio rebalancing
- Multi-exchange orchestration

**General:**
- Natural language interface (no coding required)
- Customizes to your preferences via conversation
- 29 blockchains via Privy wallets (no private key exposure)
- Agent runs 24/7 in isolated container

You want a chatbot? Fine. You want to write code? Fine. But if you want to **control your finances by trading**, Starchild has the tools custom-made for that.
